# VM Compatibility Tool — Claude/Codex 호환 안내

이 문서는 중복 상태 문서 대신 **짧은 호환 안내용 포인터**로 유지합니다.

## 우선 참조 문서

1. `docs/unified-project-plan.md`
   - 현재 프로젝트의 단일 source of truth
   - 기능 정의 / 현재 상태 / TODO / 우선순위 기준

2. `docs/stage4-system-info-auto-save-plan.md`
   - Stage 4(시스템 정보 CSV 자동 저장) 구현 전 설계안

3. `docs/windows-manual-qa-checklist.md`
   - Windows 실기 QA 체크리스트

## 문서 운영 원칙

- 기능 상태와 TODO는 `docs/unified-project-plan.md`를 기준으로 업데이트합니다.
- 구현 전 세부 설계가 필요한 작업은 `docs/` 하위에 별도 설계안으로 추가합니다.
- `README.md`는 외부 공개용 개요 문서, `CHANGELOG.md`는 배포 이력 문서로 유지합니다.

- 그다음 현재 구현 범위 Windows QA
